Project

ChromeTabber

Chrome Extension JavaScript Work problem

What it is

A tab carousel extension built because every other option I tried kept reloading pages when switching tabs, which made them useless for the thing I actually needed at work.

So I made one that keeps the rotation smooth, keeps pages alive, and solved the real problem cleanly enough that it ended up getting used where I work.

Open listing

Why it mattered

  • Fixed a real workflow problem instead of being another demo toy
  • Avoided tab reload behavior that broke competing extensions for the use case
  • Ended up implemented at work, which is a pretty decent outcome for spite-driven engineering

What it is

A rebuild of the classic Nokia Snake because I’ve been weirdly obsessed with that game forever. I kept the simple old-school loop, then added an online leaderboard so the nostalgia project could get a little competitive.

  • Keyboard controls plus mobile tap / swipe support
  • Responsive canvas with a cleaner in-site game shell
  • Online leaderboard with local fallback if the network flakes out
